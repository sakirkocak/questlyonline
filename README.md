## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/questlyonline.git
cd questlyonline
```

2. Install dependencies:
```bash
npm install
```

3. Copy environment variables:
```bash
cp env.example .env.local
```

4. Update `.env.local` with your Supabase and Typesense credentials (shared with Teknokul)

5. Run development server:
```bash
npm run dev
```

6. Open [http://localhost:3000](http://localhost:3000)

## Shared Backend

Questly shares its backend with Teknokul (Turkish education platform):

- **Supabase**: Same database, filtered by `lang='en'`
- **Typesense**: Same search index, filtered by `lang:=en`
- **Questions**: Generated from admin panel with "English" selected
